First level: 1. Как скопировать на свой компьютер репозиторий из github.com,
 что должно быть сделанно предварительно?

Ответ 1: Заходим в GHub на необходимый репрезиторий -> code -> SSH -> copy.
         Заходим с помощью терминала в папку, куда хотим клонировать 
информацию -> git clone [вставляем скопированный ранее SSH password].
         

Как создать git репозиторий из локального каталога на своем комьютере. 
Как его отправить на github.com?

Ответ2: nano file.txt[вносим изменения и сохраняем] -> git init -> git add . 
-> git commit -m "name of commit" -> git remote -v -> git push -u origin master.


Как получить на локальный компьютер изменения с github.com

Ответ3:git fetch - для вытягивания данных из удалённого репрезитория, 
без слияния.
       git pull - для вытягивания данных из удалённого репрезитория,
 со слиянием.




Дополнительно 



Какую информацию нам дают команда :
git status- показывает текущий статус содержимого папки,
            есть ли данные которые можно commit.

git log- показывает историю комитов.

git remote -v -указывает куда мы будем комитеть наш локальный репрезиторий.
